# Autoshop - Full-stack E-commerce Application

## Project Description
Autoshop is a web platform for managing vehicle parts sales. 
The application allows users to browse, search, and purchase products, 
while administrators can manage inventory and orders.


## Technologies
- **Backend:** C#, .NET Core / .NET 8+, Entity Framework Core, REST API  
- **Frontend:** Angular (TypeScript), RxJS, HTML5, CSS3 (SCSS)  
- **Database:** MS SQL Server  
- **Tools:** Git, Postman, Swagger

## Status
- Project is currently in progress  
- Backend API  has been largely implemented
- Frontend UI is under development

## Installation / Running (planned)
1. Clone the repository:git clone https://github.com/ravnjak01/Autoshop.git
2. Install .NET dependencies  
3. Install Angular dependencies  
4. Run backend and frontend servers
5. Generate data through 'dataseedgenerateendpoint' on swagger
